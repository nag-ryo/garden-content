[[抵抗]]で使用する。
数字が色に変身した言語。

4本か5本の色帯がある。

## 対応表

|**色**|**数字**|**倍率**|**誤差**|
|---|---|---|---|
|黒|0|×10⁰|–|
|茶|1|×10¹|±1%|
|赤|2|×10²|±2%|
|橙|3|×10³|–|
|黄|4|×10⁴|–|
|緑|5|×10⁵|±0.5%|
|青|6|×10⁶|±0.25%|
|紫|7|×10⁷|±0.1%|
|灰|8|×10⁸|±0.05%|
|白|9|×10⁹|–|
|金|–|×10⁻¹|±5%|
|銀|–|×10⁻²|±10%|


## 4本抵抗の読み方（左→右）


| 何本目 | 意味        |
| --- | --------- |
| 1   | 1桁目の数字    |
| 2   | 2桁目の数字    |
| 3   | 倍率(x10^n) |
| 4   | 誤差        |

※金色、銀色がある側は誤差。そこを右端にして読むのがコツ。

### 具体例
茶・黒・赤・金 の抵抗なら...

1, 0, x100, ±5%

10 x 100 = 1000Ω ±5%

## 5本抵抗の読み方（左→右）

| 何本目 | 意味              |
| --- | --------------- |
| 1   | 1桁目の数字          |
| 2   | 2桁目の数字          |
| 3   | 3桁目の数字          |
| 4   | 倍率（x10^n）       |
| 5   | 誤差（ほぼ必ず茶・赤・緑・青） |

### 具体例

茶・黒・黒・赤・茶
1, 0, 0, x100, ±1%

100 x 100 = 10,000Ω ±1%

※5本目は少し離れていることが多い。そこを右端にするのがコツ
