入力データが[[レジスタ]]内をシフトするように作られた回路。
内部では[[フリップフロップ]]回路を[[カスケード接続]]している。
[[SI]]の端子から、順番にデータを入れる。

01001010を入れると、

| 端子  | データ |
| --- | --- |
| Qa  | 0   |
| Qb  | 1   |
| Qc  | 0   |
| Qd  | 0   |
| Qe  | 1   |
| Qf  | 0   |
| Qg  | 1   |
| Qh  | 0   |

となる。
[[SCK]]がHになっているときのデータが入力される。
[[RCK]]がLだと出力を[[ラッチ]]する。Hで出力を更新する。

なので、
RCKをL → SIがL → SCKがH → SCKがL → SIがH → SCKがH → ... → RCKをH
という流れで出力する内容を決めている。
