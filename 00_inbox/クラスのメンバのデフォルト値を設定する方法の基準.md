クラスのメンバのデフォルト値を設定する方法として、以下の2通りがある。
- メンバの定義時にデフォルト値を用意する
- コンストラクタで初期化する

## メンバの定義時にデフォルト値を用意する場合
```ts
export class Test {
  private state: string = 'test';
  constructor() {
  }
}
```
このアプローチはメンバが複雑な初期化ロジックを必要としない場合や、複数のconstructorが存在せず、初期値が変更されない場合に適する。

## コンストラクタ内で初期化する場合
```ts
export class Test {

  private state: string;
  private mode: number;
  
  constructor(
	testParam: string,
	modeParam: number
  ) {
    this.state = testParam;
    if (modeParam !== 0) {
	  this.mode = modeParam * 2;
    } else {
	  this.mode = 1;
    }
  }
}
```
このアプローチはコンストラクタ内のロジックが複雑で、外部からのパラメータや条件に依存する場合に使う。
コンストラクタが複数存在する場合にも有用。
