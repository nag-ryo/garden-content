
電圧で動くスイッチ。
小さな命令で大きな電流を動かす道具でもある。

## 用語

NPNトランジスタ（2N3904 / 2N2222 など）は、足が3本。

```
   C（コレクタ） ← 仕事をする側
   |
   |
   B（ベース）   ← 命令口（ここが超重要）
   |
   |
   E（エミッタ） ← GND（帰り道）
```

覚え方：
- B = ボタン
- C = 電流の通り道
- E = 地面（GND）

## 用途例

### エアソフトガンのMOSFET

エアソフトガンは瞬間的に大電圧を流すので、トリガー部分のスイッチで火花が飛び、スイッチが焼けて使い物にならなくなることが多発する。

そこで、FETを導入して、トリガーの部分は小さい電圧で火花が飛ばないようにして寿命を伸ばすことが出来る。
