
`&`で複数の型を結合できる。
オブジェクト型とUnion型で挙動が変わる。

オブジェクトの例
```ts
interface A {
  name: string;
  lastName: string;
}

interface B {
  age: number;
  bloodType: string;
}

let person: A & B = {
  name: "John",
  lastName: "Doe",
  age: 30,
  bloodType: "A+"
};

console.log(person.name); // "John"
console.log(person.age);  // 30
```

Union型の例
```ts
// union型
type Combinable = string | number;
type Numeric = number | boolean;
type Universal = Combinable & Numeric; // 交差部分がnumberのみであるので number型になる 
```

ちなみに、key名が同じで型が違う場合はnever型になってしまう。
```ts
interface A {
	name: string;
	height: number;
}

interface B {
	age: number;
	height: string;
}

  

let person: A & B = {
	name: "John",
	age: 30,
	height: 180 // ERROR: 型 'number' を型 'never' に割り当てることはできません。
};
```
片方がnullableでもだめ。
```ts
interface A {
	name: string;
	height?: number;
}

interface B {
	age: number;
	height: string;
}

  

let person: A & B = { // ERROR: プロパティ 'height' は型 '{ name: string; age: number; }' にありませんが、型 'B' では必須です。
	name: "John",
	age: 30,
};
```
