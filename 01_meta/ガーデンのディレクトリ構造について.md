この庭の構造は、極めて単純です。

- ノートはすべて Inbox に存在する  
- 入口は Start Page（index）  
- 道しるべとして MOC（Map of Content）を置く  

フォルダは整理のためではなく、  
**読み手のための最低限の導線**としてのみ使われます。

## 一覧表

| ディレクトリ   | 目的                 |
| -------- | ------------------ |
| inbox    | 例外を除くすべてのノート       |
| _private | 非公開                |
| moc      | MOCノートを格納          |
| meta     | このサイトの説明用のノートを格納する |
